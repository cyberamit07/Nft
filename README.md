# 🛡️ PutNfts Escrow Bot

> **Secure Escrow Service for NFT Transactions on Telegram**

[![Telegram](https://img.shields.io/badge/Telegram-@PutNfts-26A5E4?style=for-the-badge&logo=telegram)](https://t.me/PutNfts)
[![Developer](https://img.shields.io/badge/Developer-@cyber_amit-FF6B6B?style=for-the-badge&logo=telegram)](https://t.me/cyber_amit)

## 📌 Features

- ✅ **Secure Escrow** - Admin verified transactions
- ✅ **Deal Management** - Create, track, and complete deals
- ✅ **Admin Panel** - Full control over all deals
- ✅ **Payment Proof** - Upload screenshots for verification
- ✅ **Dispute Resolution** - Admin mediated dispute handling
- ✅ **User Ratings** - Rate sellers after completion
- ✅ **Rich UI** - Premium inline keyboards and formatting
- ✅ **Multi-Currency** - USD, INR, BTC, ETH, USDT, BNB

## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- Telegram Bot Token (from @BotFather)

### Installation

```bash
# Clone the repository
git clone https://github.com/cyberamit/PutNfts-Escrow-Bot.git
cd PutNfts-Escrow-Bot

# Install dependencies
pip install -r requirements.txt

# Configure bot
cp .env.example .env
# Edit .env with your BOT_TOKEN

# Run bot
python src/bot.py
